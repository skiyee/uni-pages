# Uni Pages

通过一个类似 `definePageMeta` 的"函数"(类似于defineProps)，在每一个页面中定义页面参数

## 安装

```bash
pnpm add -D uni-pages

yarn add -D uni-pages

npm install -D uni-pages
```
