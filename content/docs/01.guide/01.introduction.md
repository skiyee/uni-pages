## 什么是 uni-pages？

`uni-pages` 是专为 UniApp 开发者设计的 Vite 插件，它让你可以在 Vue 单文件组件中通过 `definePageMeta` 宏直接定义页面配置，告别繁琐的 `pages.json` 手动维护。

::tip
**核心理念**：代码与配置同源，让页面配置像写组件一样自然。
::

## 为什么选择 uni-pages？

在传统的 UniApp 开发中，你需要在 `pages.json` 中手动配置每个页面的路由和样式信息。随着项目规模增大，这种方式会带来诸多问题：

- **配置分散**：页面代码和页面配置分离，维护成本高
- **容易出错**：手动同步路径容易产生拼写错误
- **类型缺失**：原生 JSON 配置无法享受 TypeScript 智能提示
- **热更新差**：修改配置后往往需要重启服务

`uni-pages` 正是为解决这些痛点而生。

## 快速预览

::code-group
```vue [pages/index/index.vue]
<script setup lang="ts">
definePageMeta({
  type: 'home',
  style: {
    navigationBarTitleText: '首页',
    enablePullDownRefresh: true,
  },
})
</script>

<template>
  <view>Hello UniApp</view>
</template>
```

```ts [vite.config.ts]
import uni from '@dcloudio/vite-plugin-uni'
import UniPages from 'uni-pages'
import { defineConfig } from 'vite'

export default defineConfig({
  plugins: [
    UniPages(),
    uni(),
  ],
})
```
::
