## 环境要求

在开始之前，请确保你的开发环境满足以下要求：

| 依赖              | 最低版本 | 推荐版本 |
|-------------------|----------|----------|
| Node.js           | 20.0.0   | 20.18.0  |
| pnpm / npm / yarn | -        | 最新版   |
| Vite              | 5.0.0    | 5.2+     |
| Vue               | 3.0.0    | 3.4+     |

::note
uni-pages 专为现代 UniApp + Vite 项目设计，支持 HBuilderX 内置编译器或 CLI 项目。
::

## 安装

使用你喜欢的包管理器安装 `uni-pages`：

::code-group
```bash [pnpm]
pnpm add -D uni-pages
```

```bash [npm]
npm install -D uni-pages
```

```bash [yarn]
yarn add -D uni-pages
```
::

## 配置 Vite

在你的 `vite.config.ts` 中添加 `uni-pages` 插件：

```ts [vite.config.ts]
import uni from '@dcloudio/vite-plugin-uni'
import UniPages from 'uni-pages'
import { defineConfig } from 'vite'

export default defineConfig({
  plugins: [
    // UniPages 必须在 uni() 插件之前
    UniPages(),
    uni(),
  ],
})
```

::warning
**重要**：`UniPages()` 必须放在 `uni()` 插件之前，否则无法正常工作。
::

## 创建配置文件

在 `src` 目录下创建 `pages.config.ts` 文件，用于定义全局页面配置：

```ts [src/pages.config.ts]
import { defineConfig } from 'uni-pages'

export default defineConfig({
  pages: [
    // pages数组中第一项表示应用启动页，也即首页
    {
      path: 'pages/index/index',
    },
  ],
  // 全局样式配置
  globalStyle: {
    navigationBarTextStyle: 'black',
    navigationBarTitleText: 'skiyee-uniapp',
    navigationBarBackgroundColor: '#F8F8F8',
    backgroundColor: '#F8F8F8',
  },
})
```

## 验证安装

安装完成后，启动开发服务器验证配置：

```bash
pnpm dev:h5
```

## 目录结构

一个典型的 uni-pages 项目结构如下：

```
your-project/
├── src/
│   ├── pages/                 # 主包页面目录
│   │   ├── index/
│   │   │   └── index.vue
│   │   └── about/
│   │       └── index.vue
│   ├── pages-sub/             # 分包页面目录（可选）
│   │   └── detail/
│   │       └── index.vue
│   ├── pages.config.ts        # 全局页面配置
│   ├── pages.json             # 自动生成，请勿手动编辑
│   ├── App.vue
│   └── main.ts
├── vite.config.ts
└── package.json
```

::caution
`pages.json` 文件由 uni-pages 自动生成，请不要手动编辑它，你的修改会在下次构建时被覆盖。
::
