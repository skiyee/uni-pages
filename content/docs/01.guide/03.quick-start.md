## 创建你的第一个页面

假设你已经按照 [安装指南](/pages/docs/guide/installation) 完成了 uni-pages 的安装，现在让我们创建第一个使用 `definePageMeta` 的页面。

### 步骤 1：创建页面文件

在 `src/pages/index/` 目录下创建 `index.vue`：

```vue [src/pages/index/index.vue]
<script setup lang="ts">
definePageMeta({
  type: 'home',
  style: {
    navigationBarTitleText: '首页',
  },
})
</script>

<template>
  <view class="container">
    <text>欢迎使用 uni-pages！</text>
  </view>
</template>

<style scoped>
.container {
  padding: 20rpx;
}
</style>
```

### 步骤 2：启动开发服务器

```bash
pnpm dev:h5
```

打开浏览器，你应该能看到页面标题显示为「首页」。

### 步骤 3：查看生成的 pages.json

uni-pages 会自动在 `src` 目录下生成 `pages.json`：

```json [src/pages.json (自动生成)]
{
  "pages": [
    {
      "path": "pages/index/index",
      "type": "home",
      "style": {
        "navigationBarTitleText": "首页"
      }
    }
  ]
}
```

## 添加第二个页面

让我们创建一个「关于」页面来体验完整流程。

```vue [src/pages/about/index.vue]
<script setup lang="ts">
definePageMeta({
  style: {
    navigationBarTitleText: '关于我们',
    navigationBarBackgroundColor: '#007AFF',
    navigationBarTextStyle: 'white',
  },
})
</script>

<template>
  <view class="about">
    <text>这是关于页面</text>
  </view>
</template>
```

保存文件后，uni-pages 会自动检测到新页面并更新 `pages.json`。无需重启服务器！
